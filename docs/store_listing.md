## 詳しい説明（4000文字以内）

🌲 森林管理アプリ - 現場作業を効率化する地図管理ツール

森林管理アプリは、林業従事者、森林組合、環境調査員のために設計された、オフライン対応の地図管理アプリケーションです。

【主な機能】

📍 作業エリア管理
• ポリゴン描画で作業エリアを正確に記録
• 面積・周長を自動計算
• 複数エリアの一覧管理
• GeoJSON/CSV形式でエクスポート

🗺️ オフライン地図
• 電波の届かない山林でも使用可能
• 地形図・航空写真の切り替え
• オフラインタイル事前ダウンロード

🌳 樹木・ポイント記録
• 個別樹木の位置をマーキング
• カスタム属性（種類、直径、高さ等）を設定
• 写真を添付して現場状況を記録

📸 フォトギャラリー
• 位置情報付き写真の撮影・保存
• 作業エリアごとの写真管理
• 複数写真の一括表示

📏 計測ツール
• 距離測定
• 面積計算
• GPS精度表示

🛤️ トラック記録
• 移動経路をGPSで自動記録
• 作業軌跡の可視化
• バックグラウンド記録対応

💾 データバックアップ
• 自動バックアップ機能
• データエクスポート（GeoJSON、CSV、KML）
• 複数プロジェクト管理

【対象ユーザー】
• 林業従事者
• 森林組合職員
• 環境調査員
• 土地管理者
• 自然保護活動家

【動作環境】
• Android 6.0以上
• GPS機能付きデバイス
• オフライン使用可能
